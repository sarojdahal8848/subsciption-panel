## Run Locally

Clone the project

```bash
  git clone https://github.com/sarojdahal8848/subsciption-panel.git
```

Go to the project directory

```bash
  cd subscription-panel
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
